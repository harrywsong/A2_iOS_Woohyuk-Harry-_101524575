## Description
An iOS application built with SwiftUI and Core Data that manages a list of 
products.

## Features
- Browse products one at a time with Previous/Next navigation
- View a full list of all products
- Search products by name or description
- Add new products to the database (10 products are seeded automatically 
on first launch)

## Project Structure
- `Persistence.swift` - Core Data stack and seed data
- `ContentView.swift` - Tab bar navigation
- `ProductDetailView.swift` - Browse products one at a time
- `ProductListView.swift` - View all products in a list
- `SearchView.swift` - Search products by name or description
- `AddProductView.swift` - Add a new product

## Requirements
- Xcode 15.2
- iOS 17.2
- Swift 5.0
